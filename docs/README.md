# Mirante4D

Mirante4D is a web viewer for 4D (3D + time) lattice light-sheet microscopy datasets.

Try it [here](https://kirchhausenlab.github.io/llsm_viewer/)!

## Program docs

- [Sparse Segmentation Hard Cutover](./sparse-segmentation/README.md)

## Install locally

Requirements:
- Node.js
- npm

```bash
npm install
npm run dev
```

The development server runs at `http://localhost:5173`.

## Production preview

```bash
npm run build
npm run preview
```
